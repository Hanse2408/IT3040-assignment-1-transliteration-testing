# IT3040 Assignment 1 - Transliteration Accuracy Testing

## Project Overview
This project contains automated Playwright test execution for testing the Chat Sinhala transliteration function available at:

https://www.pixelssuite.com/chat-translator

The test cases evaluate how accurately chat-style Singlish inputs are converted into Sinhala output.

## Tools Used
- Python
- Playwright
- OpenPyXL
- Microsoft Edge browser

## How to Install Dependencies

```cmd
pip install -U pip
pip install playwright openpyxl
